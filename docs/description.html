This package allows to build web directory using Subrion CMS.

Subrion Web Directory Script is a search engine optimized web directory software. Its user defined data field option allows to create any kind of directory from niche directories to huge portals in Yahoo and DMOZ style.

It possesses number of features significantly enhancing its abilities. Built-in payment system, unlimited nest structure of categories, featured, sponsored and partner listings are just a snapshot of its full functionality. Check the full list of features and make sure that you do not need to search further more.
